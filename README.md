# APOE Îµ4-Associated Hippocampal Atrophy Trajectories Across the Alzheimer's Disease Continuum: A Systematic Review, Meta-Analysis, and Longitudinal Validation

## Code Description

### `meta.R`
R script for systematic review and meta-analysis. Performs effect size calculations, forest plots, and subgroup analyses using the `meta` package.

### `analyze_nacc_hv.py`
Python script for longitudinal analysis of the NACC cohort. Uses linear mixed-effects models to assess hippocampal volume trajectories across APOE genotypes.

### `analyze_adni_zong.py`
Python script for ADNI cohort analysis. Investigates APOE-amyloid interactions on hippocampal atrophy using CSF biomarker stratification.
